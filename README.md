# Premium Neon Tetris

현대적이고 세련된 네온 디자인이 적용된 고전 테트리스 게임입니다.

## 주요 특징
- **다이나믹 레벨 시스템**: 20줄마다 레벨업, 10단계 속도 조절.
- **함정 블록(Trap)**: 레벨 상승 시 난이도를 높이는 랜덤 블록 생성.
- **콤보 시스템**: 연속 줄 제거 시 보너스 점수 및 시각 효과.
- **랭킹 피드백**: 로컬 스토리지를 활용한 TOP 10 랭킹 기록 서비스.
- **7-Bag 랜덤**: 모든 블록이 공정하게 출현하도록 보장하는 알고리즘.

## 실행 방법
1. `index.html` 파일을 브라우저에서 엽니다.
2. 이름을 입력하고 게임을 시작하세요!

## 조작법
- `←` `→`: 좌우 이동
- `↑`: 블록 회전 (반전 모드 적용 중)
- `↓`: 빠른 낙하
- `Space`: 하드 드롭 (즉시 낙하)
- `P`: 일시 정지
- `R`: 재시작
