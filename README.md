# Premium Neon Tetris with Gemini AI

현대적이고 세련된 네온 디자인에 **Google Gemini AI**가 결합된 차세대 고전 테트리스 게임입니다.

## 🚀 주요 특징
- **Gemini AI 개입 (AI's Whim)**: AI가 당신의 플레이를 실시간으로 해설하고, 상황에 따라 '축복'을 내리거나 '시련'을 줍니다.
- **안개 모드 (Fog Mode)**: AI의 시련으로 블록이 안개 속으로 사라집니다 (블러 효과). 15초간 집중력을 발휘하세요!
- **조작 반전 (Reverse Controls)**: AI의 장난으로 좌우 방향키가 반대로 바뀝니다. 시각적 경고가 발생하면 주의하세요!
- **AI 축복 (Blessing)**: 위기 상황에서 AI의 자비로 'I' 블록 3개가 연속으로 지급되기도 합니다.
- **다이나믹 레벨 시스템**: **10줄**마다 레벨업하며 점차 빨라지는 속도와 함정에 도전하세요.
- **7-Bag System**: 모든 블록이 공평하게 출현하도록 보장하는 표준 규칙 적용.

## 🛠 실행 및 배포 조건 (Vercel)
이 게임은 서버사이드 기능을 위해 Vercel에 최적화되어 있습니다.
1. Vercel 환경변수에 `ajou_tetris`를 추가해야 AI 기능이 정상 작동합니다.
2. 로컬에서 실행 시 `api/` 폴더를 처리할 수 있는 서버 환경이 필요합니다.

## 🎮 조작법
- `←` `→`: 좌우 이동 (반전 모드 시 주의)
- `↑`: 블록 회전
- `↓`: 빠른 낙하
- `Space`: 하드 드롭 (즉시 낙하)
- `P`: 일시 정지
- `R`: 게임 오버 후 재시작
- `S`: 시작 화면에서 시작
